# Dragon Launcher - Fast Gesture based android launcher
</p>
Dragon Launcher is a highly customizable Android launcher focused on speed and efficiency.

## Features

The main Idea of this launcher is to launch your apps via gestures.
You can customize your list of fast access apps easily in the settings, and also use the app drawer to pick an app and launch it automatically

Get the lastest version on github: [last release](https://github.com/Elnix90/Dragon-Launcher/releases/latest)

## Usage

* **Long click 3 seconds to access settings**
* Tap or long-press apps to quickly launch, view options, or uninstall.
* Customize gestures and visual settings via the Settings menu.
